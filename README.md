## I miss her bot

live: @i_miss_her_bot (Due to many security reasons, the bot might be down sometimes)

### System Requirements
1. pip
2. python >= 3.9

### Install 
1. `pip3 install -r requirements.txt`
2. `touch .env && echo "TH_USERNAME=\"your username\"" > .env && echo "TH_PASSWORD=\"your password\"" >> .env`
